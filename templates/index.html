💠 天枢·Matrix 全息操作逻辑白皮书

🚀 1. 全局文件流与指令按钮

📁 选取图纸 / 开始识别
将 PDF 上传至后端 OCR 引擎，解析完成后左侧渲染图纸，右侧自动根据图纸内容生成基础接线表格。
📥 导入 Excel
逆向工程：读取本地 Excel 文件，按照电缆编号严格切分，重新生成可编辑的交互式网格。
💾 全息矩阵存档✨ 导出选中存档
[存档]将当前页面数据以图纸名存入浏览器本地缓存(LocalStorage)。[导出]则可跨文件自由勾选存档，由后端统一打包合并为一份 Excel 下载。
当前模式：👆单选/🔲框选
控制左侧点击识别框的行为。单选可逐格填入；框选可批量框住一整排文字，系统会自动按行写入右侧表格。
🧨 格式化🗑️ 清空并重新生成
[格式化]彻底抹除浏览器内所有历史图纸数据。[清空]仅丢弃当前图纸进度并重新从 OCR 数据生成初始表格。

⚙️ 2. 单元格交互与智能逻辑

✏️ 表格修改与聚焦
点击任何单元格即可变为红色边框(选中状态)，再次双击或按任意字母键即可进入编辑模式。按 Enter 键自动跳至下一行同列位置。
🔍 全局液态搜索与替换
输入搜索词与替换词，点击[替换为]，瞬间修改当前图纸中所有匹配的文本（处于锁定🔒状态的行会被保护，免受影响）。
📋 粘贴映射 (Paste)
支持从外部 Excel 复制多行多列，直接选中起点单元格 `Ctrl+V`，数据会自动瀑布流填充。

🧠 3. 核心黑科技：联动推算与自动填充

🪄 规格裂变引擎 (电缆规格列)
当在“规格”列输入如 4x1.5(1) 时，系统会自动将该组电缆膨胀或收缩为4行，并自动编排线芯号 1~4，同时将最后1行自动标记为“备用芯”。
🔢 瀑布流连号 (回路号/端子排列)
修改某行的端子号(如 XT1 15)，其下方的行会自动递增为 XT1 16, XT1 17...。修改回路号尾数亦同。
🔄 镜像反转推算 (作用/回路号列)
输入 A相，下方两行自动生成 B相, C相。
输入 合闸位置，下方自动生成 分闸位置。
输入 +，下方自动生成 -。
🧬 历史记忆唤醒
只要输入“电缆编号”，系统会在你以前存档的图纸中疯狂检索。如果发现相同编号，将自动把历史的“回路号”和“作用”填充到当前表格！

🛡️ 4. 组级控制与最终组合渲染

🔓 锁定 / 🔒 保护
左侧的锁按钮。锁定后，该组将不再受上方的“瀑布流连号”影响，也不受全局替换影响，数据被绝对保护。
🎨 最终组合 (自动生成列)
根据填写的端排、回路等数据，自动拼接生成文本。色彩AI：若识别到跳闸/出口/电流/特定编号等敏感词，会自动变红(危险)或变黄(警告)或变绿(安全)。
